# Projeto Agência — Front-end

Projeto Front-end desenvolvido a partir de um layout base criado no Figma, com o objetivo de construir um site institucional para uma agência fictícia.

O foco do projeto foi transformar o design em código funcional, utilizando HTML, CSS e SCSS, aplicando conceitos de layout, organização visual e responsividade.

## Contexto do projeto

A página inicial foi desenvolvida com base em um layout previamente definido no Figma. A partir disso, outras seções e páginas foram adaptadas e construídas utilizando código próprio e reaproveitamento de estruturas já conhecidas.

Durante o desenvolvimento, algumas funcionalidades em JavaScript foram implementadas com auxílio de ferramentas automáticas, com foco apenas na integração e funcionamento básico.

## Minhas responsabilidades

- Implementação do layout a partir do Figma
- Estruturação do HTML semântico
- Estilização com CSS e SCSS
- Ajustes de responsividade
- Organização visual e hierarquia de informações
- Integração de scripts JavaScript básicos (com auxílio)

## O que aprendi com o projeto

- Leitura e interpretação de layouts no Figma
- Transformação de design em código
- Organização de arquivos Front-end
- Criação de layouts responsivos
- Uso de SCSS para organização e reutilização de estilos
- Uso consciente de ferramentas de apoio ao desenvolvimento

## Tecnologias utilizadas

- HTML5
- CSS3
- SCSS (Sass)
- JavaScript (uso básico)

## Observações

Este projeto tem caráter educacional e representa meu processo de aprendizado em desenvolvimento Front-end.
